# Need Feeder responsive checklist

Quick checks
- Layout: Landing, Dashboard, Donate, NGO Listing, Tracking, Profile, Transactions.
- Navigation: TopBar on desktop, SideDrawer on mobile (open/close).
- Forms: Login/Register/Donate inputs and buttons.
- Overlays: Modals and toasts stay inside viewport.
- Background: Animated layers do not overlap text.
- Timeline: Step labels wrap on small screens.

Recommended device widths
- 360 x 800 (small Android)
- 390 x 844 (iPhone 12/13)
- 414 x 896 (large iPhone)
- 768 x 1024 (tablet portrait)
- 1024 x 768 (tablet landscape)
- 1366 x 768 (small laptop)
- 1440 x 900 (desktop)
- 1920 x 1080 (desktop)

How to preview quickly
1) Open DevTools in Chrome or Edge.
2) Toggle device toolbar.
3) Pick the widths above and test navigation, forms, modals, and tracking timeline.

Known limits
- This is a frontend-only demo with mock data and no backend integration.
